# 🚀 BMW Sales Dashboard (Power BI)

## 📊 Project Overview
This Power BI dashboard analyzes BMW sales data from 2018 to 2025, providing insights into sales performance, regional trends, and top-selling models.

## 📌 Key Features
- Total Sales, Units Sold, and Average Sales KPIs  
- Year-wise sales trend analysis  
- Region-wise performance comparison  
- Top 5 models by sales  

## 🛠 Tools Used
- Power BI  
- Data Cleaning  
- Data Visualization
